# Datasheet – BBO Capstone Dataset

## Motivation
This dataset supports black-box optimisation experiments using adaptive learning strategies.

## Composition
The dataset contains:
- Weekly optimisation query results
- Reward scores
- Function evaluations
- Optimisation history

## Collection Process
The data was generated through iterative weekly submissions during the BBO capstone project.

## Preprocessing
- Filtering noisy outputs
- Aggregating optimisation scores
- Trend analysis across weeks

## Uses
- Reinforcement learning experimentation
- Adaptive optimisation research
- Sequential decision-making analysis

## Limitations
- Unknown objective functions
- Limited query budgets
- Noisy reward feedback

## Ethical Considerations
No personal or sensitive data is included.
